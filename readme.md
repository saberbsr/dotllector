# Dotllector

## Идея
- Игра вдохновлённая аркадными играми из 1980-х

## Реализация
- Библиотека arcade
- Отдельные наследованные классы для подвижных объектов (игрока, врагов)
- Разный ИИ для каждого врага
- Отдельные View для меню, игры, паузы, экрана победы и поражения

## Возможности для доработки
- Дополнительные уровни с усложнением ИИ врагов
- Сохранение результатов и таблица лидеров
